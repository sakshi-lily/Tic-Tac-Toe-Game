# 🎮 Tic-Tac-Toe Game

A simple and interactive Tic-Tac-Toe game built using modern web technologies. Play against a friend in a classic 3×3 grid and enjoy a clean, responsive user interface.

---

## 📌 Features

-  Two-player gameplay (X and O)
-  Winner detection
-  Draw detection
-  Restart game functionality
-  Responsive design for desktop and mobile
-  Clean and user-friendly interface

## 🎯 How to Play
1. Player X starts the game.
2. Players take turns marking empty cells.
3. The first player to align three marks horizontally, vertically, or diagonally wins.
